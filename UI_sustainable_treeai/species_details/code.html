menu_book 植物學描述

台灣特有種，常綠大喬木。樹皮紅褐色，呈長條片狀剝落。葉為鱗片狀，交互對生，緊貼小枝。毬果橢圓形，果鱗盾狀。為台灣著名的珍貴木材，與台灣扁柏合稱「檜木」，生長緩慢但壽命極長，常形成巨木群。其木材含有豐富精油，具防腐防蟲特性。

search 辨識指南

樹皮特徵

紅褐色，呈薄長條片狀剝落，相較扁柏更顯粗糙且裂溝較深。

葉片型態

鱗狀葉交互對生，先端銳尖，背部具不明顯的腺點。揉捏有獨特香氣。

straighten 量測注意事項

  • check_circle 樹幹基部常有顯著的板根現象，量測胸高直徑(DBH)時需避開板根影響範圍。
  • check_circle 老樹常呈中空狀，若發現樹幹有空洞，需在備註欄詳實記錄以修正材積計算。
  • check_circle 注意區分伴生樹種如台灣扁柏，可由枝葉下垂程度(紅檜較下垂)初步判斷。
